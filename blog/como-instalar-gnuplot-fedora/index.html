Como instalar o Gnuplot no Fedora

  • 06/08/2016
  • Ferramentas
  • Fedora, Ferramentas Acadêmicas, Linux

Vamos ver como instalar o Gnuplot no Fedora. Para quem não conhece, Gnuplot é uma ferramenta muito usada no meio acadêmico para plotar gráficos de funções matemáticas em 2D e 3D, sendo que a saída pode ser apresentada diretamente na tela ou exportada para arquivos em diversos formatos, como PNG, JPEG, SVG e EPS.

O Gnuplot não é disponibilizado por padrão nos repositórios oficiais do Fedora, desta forma, é necessário fazer o download do software manualmente.

Depois de fazer o download devemos abrir o terminal, entrar no diretório onde está o arquivo e descompactá-lo. Supondo que o nome do arquivo é gnuplot-5.0.3.tar.gz e está em /home/user/Downloads, entre com os comandos abaixo no terminal.

Entrar no diretório do download:


$ cd Downloads

Descompactar o arquivo de instalação:


$ tar -vzxf gnuplot-5.0.3.tar.gz

Entrar no diretório descompactado:


$ cd gnuplot-5.0.3

Agora vamos para a instalação propriamente dita. Entre com os comandos abaixo:


$ ./configure


$ sudo make install

Se não houver problemas durante esse processo, a instalação será concluída com sucesso. Para verificar se o software está funcionando, vamos fazer uma espécie de Hello Word, ou seja, vamos plotar um gráfico simples.

Para entrar no Gnuplot vá ao terminal e faça:


$ gnuplot

Com o Gnuplot aberto no terminal, vamos plotar o gráfico de sen(x):


gnuplot> plot sin(x)

Com isso, uma janela deverá ser aberta com o gráfico da função como na figura abaixo.

Eu usei o Gnuplot para gerar os gráficos do meu TCC e a qualidade ficou ótima. Para finalizar, outra dica é exportar os gráficos gerados para formatos vetoriais, como o EPS ou o SVG.
